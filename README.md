# Rest Api Node and Mongo for authentication

## Description
This is an Restful API for Node.js and Mongo. 

##### Routing         : Express
##### ODM Database    : Mongoose
##### Authentication  : Passport, JWT

## Installation

#### Donwload Code | Clone the Repo

```
git clone {repo_name}
```

#### Install Node Modules
```
npm install
```

#### Run
```
npm run start
```
